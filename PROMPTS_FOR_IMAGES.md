# Промпты для генерации изображений портфолио

## Формат файлов
- **Размер:** 800x600px (или пропорционально)
- **Формат:** WebP
- **Качество:** 80-90%
- **Стиль:** Профессиональные фотографии, реалистичные, деловые

- **Стиль:** Реалистичные профессиональные фотографии

- **Цвета:** Естественные, не перегруженные
- **Композиция:** Четкая, понятная, профессиональная
- **Настроение:** Деловое, надежное, профессиональное
